然后在浏览器中打开http://localhost:4000可以看到一个默认配置的网站,不过这个网站只能在本地打开.接下来要做的就是把它部署到服务器了.

将网站部署到服务器

  • 如果你是GitHub新手,可以通过这个官方提供的HELLO WORLD熟悉一下GitHub的基本操作.

    初步了解基本的操作后,点击你主页右上角的加号, 创建(Create)一个 Repository .在 Repository name下面填写 username.github.io,这里的username就是你GitHub的用户名,不能自定义!

    接下来打开用编辑器你工作目录里的 _config.yml 文件,在最下面的 “deploy:” 下填写以下内容:
    type: git
    respository: https://githun.com/username/username.GitHub.io.git
    branch: master
    (其中的username即为你GitHub的用户名)
    (注意!这三个地方的冒号: 后面都是要有一个空格的,这是yaml的语法要求)

  • 然后,如果你刚刚的 hexo server 还在运行的话要先 crl+z 终止,然后我们输入以下命令来将我们本地的网站部署到GitHub上.
